# ⚡ AI 투자 자문 V2 - 실전형 액션 중심 업그레이드

## 🚨 문제점 (Before)

사용자 피드백:
> "AI 분석 결과가 너무 일반적인 얘기만 하고, 정작 내 종목에 대한 구체적인 매매 신호가 없어서 실망스러웠어."

### 기존 문제
- ❌ "반도체 업황이 개선되고 있습니다" - 뻔한 시황 설명
- ❌ "시장 변동성에 유의하세요" - 추상적 조언
- ❌ 일반론만 나열, 실제 액션 없음
- ❌ 종목별 "팔지/말지" 명확한 답변 부재

---

## ✅ 해결책 (After)

### 새로운 AI 페르소나
**"수익률 TOP 1% 실전 투자 자문가"**
- 두루뭉술한 시황 설명 금지
- 오직 **"이 종목을 지금 팔아야 하는가? 말아야 하는가?"**에 명확한 답변
- 투자자가 바로 실행할 수 있는 구체적 액션 제시

---

## 🎯 핵심 변경사항

### 1. 일반론 금지 규칙
```
❌ "반도체 업황이 개선되고 있습니다"
❌ "시장 변동성에 유의하세요"

✅ "SK하이닉스 (현재 +443%): KB증권 리포트 목표가 25만원 제시,
   HBM 수요 증가 확인. → 홀딩 유지, 50% 분할 매도 고려"
```

### 2. 근거 필수 제시
모든 판단에 아래 중 하나 이상 인용:
- **리포트 내용**: "KB증권 리포트에서 SK하이닉스 목표가 25만원 제시"
- **거시지표**: "VIX 14로 안정권, 금리 동결 유지"
- **기초자산 추세**: "나스닥 상승 돌파, TQQQ 추세 양호"

### 3. 개별 종목별 명확한 액션
각 보유 종목마다 반드시 아래 중 하나 제시:

| 액션 | 의미 | 예시 |
|------|------|------|
| ✅ **홀딩 (Hold)** | 현재 포지션 유지 | 추세 양호, 리포트 호재 |
| ⚠️ **비중 축소 (Reduce)** | 30~50% 매도 | 과열 신호, 부분 수익 실현 |
| 🔴 **전량 매도 (Sell All)** | 즉시 전량 청산 | 추세 붕괴, 악재 출현 |
| ⏳ **관망 (Wait & See)** | 다음 신호 대기 | 방향성 불명확, 중립 구간 |

### 4. 레버리지 ETF 특별 규칙
```
✅ 기초 자산(나스닥/S&P500) 상승 추세 → 홀딩 OK
⚠️ VIX 20 이상 → 비중 축소 권장
🔴 횡보/하락 추세 → 전량 매도 권장
```

## 📋 새로운 분석 구조

### 🚨 오늘의 핵심 액션 (최우선)

#### 1. 🔴 즉시 매도 권장
손실/수익 무관, 리포트/지표 기반으로 **지금 팔아야 할 종목**

```markdown
- **덱스터 (-61.72%)**
  - **판단 근거**: 실적 악화 지속, 리포트 없음, 회복 가능성 낮음
  - **액션**: 전량 매도
  - **타이밍**: 즉시 (추가 하락 전)
```

#### 2. ✅ 홀딩 유지 (팔지 마세요)
현재 보유 지속이 최선인 종목

```markdown
- **SK하이닉스 (+443.75%)**
  - **판단 근거**: [IBKS Daily] 반도체 업황 개선 기대, HBM 수요 증가
  - **홀딩 이유**: 상승 추세 유지, 리포트 긍정적
  - **목표가**: 25만원 (KB증권 제시)
```

#### 3. ⏳ 관망 (추가 신호 대기)
지금 당장 액션 불필요, 다음 신호 확인 후 결정

```markdown
- **TQQQ (-5.23%)**
  - **판단 근거**: 나스닥 중립 구간, VIX 16 (보통 수준)
  - **대기 이유**: 방향성 불명확
  - **주시할 신호**: VIX 20 돌파 여부, 나스닥 추세 전환
```

---

### 💰 추가 매수 기회
지금 사거나, 보유 종목 추가 매수

```markdown
- **SK하이닉스**
  - **매수 근거**: 리포트 호재 지속, 저평가 구간
  - **매수 비중**: 전체 자산의 5%
  - **목표가**: 25만원
```

---

### 📊 포트폴리오 종합 진단

#### 위험 수준
- 🟢 안정 / 🟡 보통 / **🔴 고위험** ← 하나 선택
- **이유**: 레버리지 ETF 비중 40%, 섹터 쏠림 심함

#### 리밸런싱 필요 여부
- **필요** / 불필요 ← 하나 선택
- **조정 방향**: 레버리지 40% → 20%로 축소

#### 현금 비중 권장
- 추천 현금 보유 비율: **30%**
- **이유**: 변동성 증가 대비, 매수 기회 포착용

---

### 🎯 최종 액션 체크리스트

```
- [ ] **즉시 매도할 종목**: 덱스터, 삼성SDI
- [ ] **홀딩 유지할 종목**: SK하이닉스, 삼성전자
- [ ] **추가 매수 고려**: SK하이닉스 (5% 비중)
- [ ] **현금 비중 조정**: 10% → 30%
```

## 🚀 사용 방법

1. **학습(📚) 탭** → **오늘의 분석** 클릭
2. **🧠 AI 심층 분석 시작** 버튼 클릭
3. **10-30초 대기** (AI가 분석 중)
4. **결과 확인**:
   - 🔴 즉시 매도할 종목 확인
   - ✅ 홀딩 유지할 종목 확인
   - 🎯 최종 액션 체크리스트 실행
5. **학습 메모 작성** 후 저장
