# 포트폴리오 초기화 문제 해결

## 문제 상황
- 포트폴리오가 예시 데이터로 초기화됨
- 구글 시트와 연동한 데이터가 사라짐

## 해결 방법

### 방법 1: 브라우저 콘솔에서 localStorage 확인

1. 브라우저에서 **F12** (개발자 도구 열기)
2. **Console** 탭 클릭
3. 다음 명령어 입력하여 현재 저장된 포트폴리오 확인:

```javascript
localStorage.getItem('insight-log-portfolio')
```

4. 만약 `null`이거나 예시 데이터라면, 구글 시트에서 다시 가져와야 합니다.

### 방법 2: 포트폴리오 탭에서 새로고침

1. **포트폴리오(Portfolio)** 탭으로 이동
2. 오른쪽 상단의 **새로고침 버튼** 클릭 (🔄 아이콘)
3. 구글 시트에서 최신 데이터를 가져옵니다

### 방법 3: 수동으로 종목 추가

포트폴리오 탭에서:
1. **+ 종목 추가** 버튼 클릭
2. 각 종목 정보 입력
3. 저장

### 방법 4: localStorage 완전 초기화 (최후의 수단)

브라우저 콘솔에서:

```javascript
// 모든 앱 데이터 삭제
localStorage.clear()

// 페이지 새로고침
window.location.reload()
```

그 다음:
1. 포트폴리오 탭에서 새로고침 버튼 클릭
2. 구글 시트 데이터가 자동으로 로드됩니다

## 예상 원인

1. **브라우저 캐시 문제**: 페이지를 강제 새로고침 (Ctrl+Shift+R / Cmd+Shift+R)
2. **개발 서버 재시작**: 서버를 재시작하면 초기 상태로 돌아갈 수 있음
3. **localStorage 손상**: 브라우저 설정이나 프라이빗 모드에서 localStorage가 제대로 작동하지 않을 수 있음

## 임시 해결책

포트폴리오 탭의 **새로고침 버튼**을 클릭하면 구글 시트에서 최신 데이터를 가져옵니다!
